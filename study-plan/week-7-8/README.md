   - Check all requirements

### Day 11-14: Final Review

1. **Knowledge Check**
   - Review all topics

   - Refresh exploit techniques

2. **Mental Preparation**
   - Rest before exam
   - Prepare environment
   - Test tools

## Exam Tips

1. **Time Management**
   - 24h đầu: Code review và exploit development
   - 24h sau: Report writing và verification

2. **Documentation**
   - Take screenshots immediately
   - Document every step
   - Save all exploit code

3. **Verification**
   - Test exploits multiple times
   - Verify all requirements met
   - Check report completeness

4. **Common Mistakes**
   - Not reading source code thoroughly
   - Missing vulnerability chains
   - Incomplete reports
   - Not testing exploits
